## SecretShopBot-E7 v1.2.1

### 주요 변경 사항

- Windows 제목 표시줄의 앱 아이콘 위치에 `top_icon`이 표시되도록 조정했습니다.
- `top_icon` 이미지의 배경을 제거해 제목 표시줄 아이콘으로 자연스럽게 보이도록 정리했습니다.
- Windows 실행 파일에 회사명, 제품명, 파일 설명, 파일 버전, 제품 버전 메타데이터를 추가했습니다.
- 릴리즈 스크립트가 `v1.2.1` 형식의 버전 표기를 기본으로 사용하도록 정리했습니다.
- 실행 파일 아이콘과 버전 리소스를 빌드 시점에 자동 생성하도록 개선해 릴리즈 산출물 일관성을 높였습니다.
- 기존 `v1.2.0`의 아이콘 변경과 GUI 문구/효율 표기 개선 사항을 유지한 상태로 재패키징했습니다.

### 오탐 완화 관점에서의 변경

- 실행 파일에 비어 있던 Windows 버전 정보를 채워 보안 제품이 식별할 수 있는 기본 메타데이터를 보강했습니다.
- 배포 버전명을 태그와 산출물에서 `v` 접두사 형식으로 통일했습니다.
- 다만 현재 실행 파일은 여전히 코드 서명되지 않았으므로, 일부 환경에서는 오탐 가능성이 남아 있을 수 있습니다.
