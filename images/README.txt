# 필요한 이미지 파일 목록

## 아이템 이미지 (images/items/)
- mystic_medal.png: 신비의 메달 이미지
- covenant_bookmark.png: 성약의 책갈피 이미지

## 버튼 이미지 (images/buttons/)
### 상점 갱신 프로세스 (갱신 -> 확인 2단계)
- refresh_button.png: 갱신 버튼 (상점 리프레시 시작)
- confirm_button.png: 확인 버튼 (갱신 확인)

### 아이템 구매 프로세스 (구입 -> 구매 2단계)
- purchase_button.png: 구입 버튼 (1단계)
- buy_button.png: 구매 버튼 (2단계, 최종 구매)
- purchase_button_disabled.png: 비활성화된 구입 버튼 (구매 완료 후 검증용)

## 이미지 준비 방법
1. 게임 실행 후 비밀상점 화면에서 각 요소를 스크린샷으로 캡처
2. 이미지 편집 도구로 필요한 부분만 잘라내기
3. 위 경로에 맞게 이미지 파일 저장
4. PNG 형식 권장

## 팁
- 이미지는 가능한 한 작고 명확하게 캡처
- 배경이 포함되지 않도록 정확히 잘라내기
- purchase_button_disabled.png는 아이템 구매 후 선명도가 떨어진 상태의 구입 버튼을 캡처
  (이 이미지로 구매 완료를 검증하여 골드 부족 시 갱신을 방지)
