# SecretShopBot-E7

에픽세븐의 반복 작업을 도와주는 Windows용 자동화 도구입니다. 앱플레이어 화면을 인식해 비밀상점 갱신, 펭귄 구매, 장비 옵션 리롤을 실행합니다.

## 주요 기능

- 비밀상점에서 신비의 메달, 성약의 책갈피, 우정 포인트를 찾고 구매
- 장비 옵션 리롤 화면에서 원하는 옵션과 수치가 나올 때까지 반복
- 펭귄 구매 화면에서 지정한 횟수만큼 구매 진행
- 여러 앱플레이어를 세션별로 나누어 관리
- 실행 로그와 구매/리롤 통계를 화면에서 확인

## 사용 전 준비

- Windows 10 이상을 권장합니다.
- 앱플레이어 해상도는 `1280x720`, DPI는 `240` 기준을 권장합니다.
- 앱플레이어에서 ADB 디버깅을 켜야 합니다.
- 에픽세븐은 자동화하려는 화면까지 직접 열어 둔 상태에서 시작해야 합니다.

## 다운로드와 실행

1. 최신 릴리즈 페이지로 이동합니다.

```text
https://github.com/pelierze/SecretShopBot-E7/releases/latest
```

2. `SecretShopBot-E7-v버전.zip` 파일을 다운로드합니다.
3. 원하는 폴더에 압축을 풉니다.
4. 압축을 푼 폴더 안의 `SecretShopBot-E7.exe`를 실행합니다.

처음 실행할 때 Windows SmartScreen 또는 백신 경고가 표시될 수 있습니다. 릴리즈 출처를 확인한 뒤 실행하세요.

## 앱플레이어 ADB 설정

앱플레이어 설정에서 ADB 디버깅을 켜 주세요.

- BlueStacks: 설정의 고급 설정에서 Android Debug Bridge를 켭니다. 기본 포트는 보통 `5555`입니다.
- LDPlayer: 설정의 기타 설정에서 ADB 디버깅을 켭니다. 기본 포트는 보통 `5555`입니다.
- Nox: 설정에서 Root 또는 ADB 관련 옵션을 켭니다. 기본 포트는 보통 `62001`입니다.

앱플레이어가 실행 중이어야 프로그램의 `장치 검색`에서 찾을 수 있습니다.

## 기본 사용 순서

1. 앱플레이어에서 에픽세븐을 실행합니다.
2. 사용할 기능에 맞는 게임 화면을 미리 열어 둡니다.
3. `SecretShopBot-E7.exe`를 실행합니다.
4. `장치 검색`을 누릅니다.
5. 목록에서 사용할 앱플레이어를 선택합니다.
6. IP 주소와 포트가 채워지면 `연결`을 누릅니다.
7. 사용할 탭을 선택하고 필요한 값을 입력합니다.
8. `시작`, `리롤 시작`, 또는 `펭귄런 시작`을 눌러 실행합니다.
9. 중단하고 싶을 때는 각 탭의 중지 버튼을 누릅니다.

## 비밀상점 갱신

비밀상점 화면을 열어 둔 뒤 `비밀상점` 탭에서 사용합니다.

1. 구매할 항목을 선택합니다.
2. 리프레시 횟수를 입력합니다.
3. 구매 완료 검증 횟수를 확인합니다.
4. 이미지 인식이 불안정하면 매칭 정확도를 조금 조정합니다.
5. `시작`을 누릅니다.

프로그램은 현재 상점 화면에서 목표 아이템을 찾고, 필요한 경우 구매 버튼을 눌러 구매합니다. 현재 페이지에서 목표 아이템을 찾지 못하면 다음 페이지로 스크롤하고, 다음 회차가 남아 있으면 상점을 갱신합니다.

## 장비 옵션 리롤

장비 옵션 리롤 화면을 열어 둔 뒤 `장비 리롤` 탭에서 사용합니다.

1. 목표 옵션을 선택합니다.
2. 목표 수치를 입력합니다.
3. 여러 옵션을 동시에 볼 경우 중지 방식을 선택합니다.
4. 최대 리롤 횟수를 입력합니다.
5. 리롤 전 대기 시간을 확인합니다.
6. `리롤 시작`을 누릅니다.

지원하는 옵션은 속도, 공격력, 생명력, 방어력, 치명타 확률, 치명타 피해, 효과저항, 효과적중입니다.

## 펭귄 구매

펭귄 구매 화면을 열어 둔 뒤 `펭귄` 탭에서 사용합니다.

1. 구매 사이클 횟수를 입력합니다.
2. `펭귄런 시작`을 누릅니다.
3. 진행 상황은 로그와 통계 영역에서 확인합니다.

프로그램은 펭귄 항목과 구매 버튼을 찾고, 구매 팝업에서 최대 수량과 최종 구매 버튼을 차례대로 누릅니다.

## 이미지 테스트

화면 인식이 잘 되는지 확인하고 싶을 때 `이미지 테스트`를 사용하세요.

- 현재 앱플레이어 화면에서 등록된 이미지가 인식되는지 확인합니다.
- 인식이 안 되면 해상도, DPI, 게임 화면 위치를 먼저 확인하세요.
- 그래도 맞지 않으면 매칭 정확도를 조금 낮춰 테스트할 수 있습니다.

## 여러 세션 사용

프로그램은 `세션 1`, `세션 2`처럼 여러 세션을 제공합니다.

각 세션은 연결된 앱플레이어, 실행 상태, 로그, 통계를 따로 관리합니다. 여러 앱플레이어를 동시에 쓰려면 세션마다 다른 장치를 선택해 연결하세요.

## 생성되는 파일

실행 중 필요한 로그와 임시 화면 캡처는 `logs` 폴더에 저장될 수 있습니다.

이 파일들은 문제 확인용이며, 평소에는 직접 수정할 필요가 없습니다.

## 자주 생기는 문제

### 장치 검색에 앱플레이어가 보이지 않는 경우

- 앱플레이어가 실행 중인지 확인하세요.
- 앱플레이어의 ADB 디버깅이 켜져 있는지 확인하세요.
- 포트가 맞는지 확인하세요. 일반적으로 BlueStacks와 LDPlayer는 `5555`, Nox는 `62001`을 사용합니다.
- 방화벽이나 백신이 ADB 연결을 막고 있지 않은지 확인하세요.

### 연결은 됐지만 버튼이나 아이템을 찾지 못하는 경우

- 해상도가 `1280x720`인지 확인하세요.
- DPI가 `240`인지 확인하세요.
- 에픽세븐 화면이 자동화하려는 화면에 정확히 열려 있는지 확인하세요.
- `이미지 테스트`로 현재 화면에서 이미지가 인식되는지 확인하세요.
- 매칭 정확도를 조금 낮춰 다시 시도하세요.

### 구매 후 멈추는 경우

- 게임 재화가 부족하지 않은지 확인하세요.
- 구매 완료 팝업이나 확인 버튼이 화면에 가려져 있지 않은지 확인하세요.
- 앱플레이어 배율, 해상도, 언어 설정이 기본 이미지와 크게 다르지 않은지 확인하세요.

## 사용 시 주의사항

- 이 프로그램은 개인 사용을 위한 자동화 도구입니다.
- 게임 이용 약관을 확인한 뒤 사용 여부를 직접 판단하세요.
- 자동화 도구 사용으로 발생하는 계정 제재나 손실은 사용자 책임입니다.
- 처음 사용할 때는 리프레시 횟수나 리롤 횟수를 작게 설정해 테스트하는 것을 권장합니다.
